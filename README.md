# Dev\_Scaffold

The Scaffold Protocol is a human-in-the-loop, YAML-based, step-by-step development workflow for .NET projects. 
The goal of the protocol is to enable an experienced senior/lead developer to carry out development tasks alone, 
in a controlled and auditable manner – within a competitive development timeframe.

The core principle: \*\*AI is a tool, not an orchestrator.\*\* Every step's output goes through human validation before being passed 
as input to the next step.
